SSH使用指南

SSH使用指南

连接方法

使用如下命令连接至远程机器

ssh user@remote -p port

  • user 是你在远程机器上的用户名，如果不指定的话默认为当前用户
  • remote 是远程机器的地址，可以是 IP或域名
  • port 是 SSH Server 监听的端口，如果不指定的话就为默认值 22

安装 OpenSSH Server

如果遇到了上面的提示,说明远程机器上可能没有安装 OpenSSH Server

执行命令sudo apt-get install openssh-server安装 OpenSSH Server (需要sudo权限)

免密码登入

通过设置SSH钥匙并将公钥上传至远程机器,我们可以实现免密码登入.

绝大多数Linux发行版都预装了ssh-copy-id ,MacOS可通过brew install ssh-copy-id命令快速安装.
